## ⚙ 주요 기능

- 프로젝트 폴더에서 다수의 파일 선택
- 우클릭 메뉴에서 “Copy Multi File Contents” 실행
- 파일 제목과 함께 각 파일의 내용이 구분선과 함께 복사됨
- 클립보드에 자동 저장됨
- 각각 파일마다 구분할 수 있는 구분선 === file.name === 추가
